# meu-primeiro-app-dio
## Repositório feito nas aulas do curso: "Criando um App Android com Suporte a Vários Idiomas"

### Nesse repositório, foi feito a internacionalização para os seguintes idiomas:
- *Português (pt);*
- *Inglês (en);*
- *Espanhol (es);*
- *Italiano (it);*
- *Japonês (ja).*
